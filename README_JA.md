# Gemini Mode Auto-Enabler

Google Gemini AIのチャット画面で、お好みのモード（思考モードまたはProモード）を自動的に有効化するChrome拡張機能です。

## 機能

- Geminiページを開くと自動的に選択したモードを有効化
- ポップアップから **思考モード**・**Proモード**・**無効** を選択可能
- 設定はChrome ストレージに保存され、デバイス間で同期
- 新しいチャット開始時も自動でモードを有効化
- すでに対象モードが選択済みの場合は何もしない
- ユーザーの手動切り替えを尊重（同一チャット内での再切り替えなし）
- 英語・日本語UIに対応

## 使い方

ツールバーの拡張機能アイコンをクリックすると設定ポップアップが開きます。希望のモードを選択してください。

| モード | 説明 |
|---|---|
| 思考モード | より深く考えてから回答する思考モードを自動で有効化 |
| Pro モード | 最高性能のモデルを使用するProモードを自動で有効化 |
| 無効 | 自動切り替えをしません |

### 開発版をインストール
1. このリポジトリをクローンまたはダウンロード
2. 依存関係をインストールしてビルド:
   ```bash
   npm install
   npm run build
   ```
3. Chrome で `chrome://extensions` を開く
4. 右上の「デベロッパーモード」をONにする
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. `dist/` フォルダを選択

## 対象ユーザー

- Gemini AI Pro プランをご利用の方
- 常に特定のモード（思考モードまたはProモード）で使いたい方

## 注意事項

- Google Gemini のUIが変更された場合、動作しなくなる可能性があります
- 思考モード・Proモードの利用には Gemini Advanced サブスクリプションが必要です
