# Gemini 思考モード自動有効化

Google Gemini AIのチャット画面で自動的に「思考モード」を有効化するChrome拡張機能です。

## 機能

- Geminiページを開くと自動的に「思考モード」を選択
- 新しいチャット開始時も自動で「思考モード」を有効化
- 既に思考モードの場合は何もしない
- ユーザーの手動切り替えを尊重（同一チャット内での再切り替えなし）

### 開発版をインストール
1. このリポジトリをクローンまたはダウンロード
2. 依存関係をインストールしてビルド:
   ```bash
   npm install
   npm run build
   ```
3. Chrome で `chrome://extensions` を開く
4. 右上の「デベロッパーモード」をONにする
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. `dist/` フォルダを選択

## 対象ユーザー

- Gemini AI Pro プランをご利用の方
- 常に「思考モード」で回答を得たい方

## 注意事項

- Google Gemini のUIが変更された場合、動作しなくなる可能性があります
- 英語・日本語UIに対応しています
