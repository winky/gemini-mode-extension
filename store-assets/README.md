## Store Listing Information

### Name (max 45 characters)
Gemini Thinking Mode Auto-Enabler

### Summary (max 132 characters)
Automatically enables "Thinking Mode" on Google Gemini AI chat. No more manual switching for every new conversation.

### Detailed Description
```
A simple utility for Gemini AI Pro plan users.

【Features】
• Automatically selects "Thinking Mode" when opening Gemini
• Re-enables on new chat sessions
• Skips if already in Thinking Mode
• Respects manual mode changes within the same chat

【Who is this for?】
• Users who prefer "Thinking Mode" responses by default
• Those who want to skip the manual mode switching

【Privacy】
This extension does NOT collect any user data.

【Supported Languages】
English and Japanese UI
```
