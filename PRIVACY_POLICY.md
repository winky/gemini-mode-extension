# Privacy Policy - Gemini Thinking Mode Auto-Enabler

**Last Updated:** 2026-02-06

## Overview

"Gemini Thinking Mode Auto-Enabler" is a Chrome extension that automatically enables "Thinking Mode" on Google Gemini AI.

## Data Collection

**This extension does NOT collect, store, or transmit any user data.**

Specifically, we do NOT collect:
- Personal information
- Browsing history
- Chat content or conversations
- Cookies or session data
- Any form of analytics or telemetry

## How the Extension Works

The extension operates entirely within your browser and:
1. Detects when you open the Gemini chat page (gemini.google.com/app)
2. Automatically clicks the "Thinking Mode" option in the mode dropdown
3. All processing happens locally in your browser

## Permissions Used

- **Host Permission (gemini.google.com)**: Required to run the content script that detects and clicks the mode selection button. The extension ONLY operates on Gemini pages.

## Third-Party Services

This extension does not communicate with any external servers or third-party services.

## Changes to This Policy

Any changes to this privacy policy will be posted on the Chrome Web Store listing and in this document.
